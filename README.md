# Minimal Snackbar

A beautiful minimal top snackbar for Flutter.

## Features

- Success snackbar
- Failure snackbar
- Info snackbar
- Top positioned
- Clean UI

## Usage

```dart
import 'package:minimal_snackbar/minimal_snackbar.dart';

MinimalSnackbar.showSuccess(context, "Saved!");
MinimalSnackbar.showFailure(context, "Error!");
MinimalSnackbar.showInfo(context, "Information");
```
